设置角色

设置角色

角色的概念为各位教师所担任的职务，任课教师``班主任``学工部等等。

总体步骤：

  • 点击下方按钮>选择设置角色>选择担任角色后提交

  • 若为班主任还需设置负责班级>点击加号选择班级>如需移除班级 点击班级名>提交

注意事项： 系统根据设置角色派发学生请假申请，请教师们务必设置角色,否则无法审核所负责请假申请，谢谢。图中信息仅为测试信息。

1. 点击下方按钮设置角色

2. 选择设置角色

3. 选择担任角色后提交

4. 若为班主任请设置负责班级

5. 点击加号选择班级

6. 如需移除班级 直接点击班级名

7. 提交
